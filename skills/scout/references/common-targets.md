# Common Scout Targets

## JavaScript/TypeScript Libraries

### React Ecosystem
- `facebook/react` — Core React source
- `vercel/next.js` — Next.js implementation
- `tailwindlabs/tailwindcss` — Tailwind internals

### Node.js
- `expressjs/express` — Express middleware patterns
- `fastify/fastify` — Fastify plugin system
- `prisma/prisma` — Prisma schema engine

### Build Tools
- `vitejs/vite` — Vite plugin API
- `webpack/webpack` — Webpack loader/processor patterns
- `esbuild/esbuild` — Bundler internals

## Python Libraries

### Data Science
- `pandas-dev/pandas` — DataFrame internals
- `numpy/numpy` — Array operations
- `scikit-learn/scikit-learn` — ML patterns

### Web
- `tiangolo/fastapi` — FastAPI dependency injection
- `psf/requests` — HTTP client patterns
- `sqlalchemy/sqlalchemy` — ORM internals

## Go Libraries

### Web/Frameworks
- `gin-gonic/gin` — Gin router patterns
- `go-chi/chi` — Chi middleware
- `golang/golang` — Standard library patterns

### CLI
- `spf13/cobra` — Command patterns
- `spf13/viper` — Configuration patterns

## Rust Libraries

- `tokio-rs/tokio` — Async runtime
- `serde-rs/serde` — Serialization patterns
- `actix/actix-web` — Web framework

## Query Strategy

1. **Start with README** — understand purpose and API
2. **Check src/ structure** — find relevant modules
3. **Search for keywords** — use grep for specific functions
4. **Read tests** — understand expected behavior
5. **Check issues/discussions** — find known gotchas
