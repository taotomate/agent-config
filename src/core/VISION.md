# Taotomate Vision & Core Architecture

Este documento define los lineamientos fundacionales del ecosistema de agentes de Taotomate. Toda la lógica operativa del agente (`agents.md`) se subordina a esta visión.

## Notas e Inspiración
Este framework es un desarrollo original de **Taotomate**, fuertemente inspirado a nivel conceptual, arquitectónico (SDD) y de herencia de identidad en el ecosistema "Gentleman AI" de Alan Buscaglia. Se reconoce y agradece su valioso aporte open source, el cual sentó bases metodológicas que luego fueron adaptadas a las necesidades y filosofía de este repositorio.
